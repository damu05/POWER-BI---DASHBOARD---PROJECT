# POWER-BI---DASHBOARD---PROJECT
INSURANCE RISK AND CLAIM ANALAYSIS

## OBJECTIVES

•	Analyze insurance data to identify trends and patterns
•	Track key performance indicators (KPIs)
•	Enable data-driven decision-making through interactive visuals
•	Demonstrate Power BI skills for portfolio purposes

### TOOLS&TECHNOLOGIES USED

• Power BI Desktop
•	Microsoft Excel / CSV (Data Source)
•	DAX (Data Analysis Expressions)
•	Power Query


  #### DATASET INFORMATION

  •	Domain: Insurance
	•	Data includes:
	•	Policy details
	•	Customer information
	•	Claims data
	•	Premium and revenue data

  ##### DASHBORAD FEATURES

  •	KPI cards for total policies, claims, and revenue
	•	Year-wise and region-wise analysis
	•	Policy type and customer segmentation
	•	Interactive slicers and filters
	•	Clean and professional dashboard layout

  1.	Download the .pbix file from this repository

	4.	Explore the dashboard using filters and slicers

#### LEARNING AND KEY TAKEAWAYS

•	Data cleaning and transformation using Power Query
•	Creating calculated columns and measures using DAX
•	Designing user-friendly dashboards
•	Understanding insurance business metrics
